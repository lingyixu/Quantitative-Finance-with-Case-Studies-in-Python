# Quantitative-Finance-with-Case-Studies-in-Python
Repo for code examples in Quantitative Finance with Case Studies in Python by Chris Kelliher

**Note: This book is the second edition of _Quantitative Finance with Python_. For readers of the first edition, please refer to [this repo](https://github.com/lingyixu/Quant-Finance-With-Python-Code) instead.**

## Prerequisites
As of January 2026, the notebooks in this repository have been tested with:

- Python 3.10+
- pip or conda

Required Python packages (with pinned versions) are listed in `requirements.txt`.
